# 🎯 Juego del Número Secreto

## 📖 Descripción

El **Juego del Número Secreto** es una aplicación web interactiva donde el jugador pone a prueba su intuición intentando adivinar un número generado aleatoriamente dentro de un rango determinado.

Después de cada intento, el juego proporciona pistas indicando si el número secreto es **mayor** o **menor** que el número ingresado, hasta que el jugador lo adivine.

---

## 🚀 Características

- 🎲 Generación aleatoria del número secreto.
- 🔢 Ingreso de intentos por parte del usuario.
- 💡 Pistas para ayudar al jugador.
- 🏆 Mensaje de felicitación al ganar.
- 🔄 Opción para iniciar una nueva partida.

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

---

## ▶️ Cómo jugar

1. Abre el juego en tu navegador.
2. Ingresa un número dentro del rango indicado.
3. Haz clic en **Intentar**.
4. Lee la pista que aparece:
   - 📈 El número secreto es mayor.
   - 📉 El número secreto es menor.
5. Sigue intentando hasta encontrar el número correcto.
6. Si ganas, podrás comenzar una nueva partida.
